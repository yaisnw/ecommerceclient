## Getting Started

### Prerequisites
- Node.js 20+

### Installation

```bash
git clone https://github.com/yaisnw/ecommerceclient.git
cd ecommerceclient
npm install
```

### Environment Variables

Create a `.env` file in the root directory:

```env
REACT_APP_BACKEND_URL=
```

### Running Locally

```bash
# Development
npm start

# Production build
npm run build
```
